# EVM

### **Cross-Network Stablecoin Transfers**

When a user wants to transfer their stablecoins from one network to another, VAULT Wallet employs **Chainlink CCIP** (Cross-Chain Interoperability Protocol) for secure and reliable transfers. This method is particularly relevant for USDC tokens, as Chainlink CCIP is well-suited for handling native token transfers across multiple blockchains.

In cases where the user is utilizing stablecoins other than USDC, a pre-conversion process is executed using the **ODOS protocol**. ODOS Smart Order Routing facilitates the conversion of these stablecoins into USDC prior to executing the cross-chain transfer. This approach allows users to seamlessly move assets between networks, regardless of the original stablecoin being used, while ensuring the entire process is efficient and cost-effective.

### **Chainlink Commission and Transaction Process**

The commission for Chainlink's CCIP service is applied at the moment of sending tokens. To facilitate this, VAULT Wallet automatically purchases **LINK tokens** required for the transaction on a third-party decentralized exchange (DEX), such as **Uniswap**. This seamless integration ensures that users do not need to manually acquire LINK tokens to pay for the Chainlink service; instead, the required LINK tokens are acquired automatically, simplifying the user experience.

By leveraging Chainlink CCIP and ODOS protocol, VAULT Wallet provides a highly streamlined and user-centric solution for cross-network stablecoin transfers, further enhancing the flexibility and utility of the wallet. This integration minimizes the manual steps required by the user, delivering a cohesive and optimized experience for interacting across blockchain ecosystems.\
