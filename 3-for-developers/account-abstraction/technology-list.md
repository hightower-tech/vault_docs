# Technology List



1. **ZeroDev**: One of the most trusted solutions in Account Abstraction, powering more than 50% of all AA wallets and supporting a significant portion of DeFi volume that leverages AA.
2. **Chainlink CCIP**: A blockchain interoperability protocol that enables developers to build secure applications capable of transferring tokens, messages (data), or both tokens and messages across chains.
3. **Cross-Chain Transfer Protocol (CCTP)**: A permissionless on-chain utility that facilitates secure USDC transfers between blockchains via native burning and minting.
4. **ODOS Smart Order Routing**: Provides smart order routing across multiple blockchain protocols, utilizing over 900 liquidity sources and thousands of token pairs to deliver ultimate savings to users.
